The KrafterSMP

...is a Minecraft (Bedrock) survival server (More like HermitCraft than the Dream SMP) which encourages players to engage with one another, creating village, towns, shops, roads, economies, etc. The goal of this server is to create a fun environment for people to simply have fun gaming. The server does cap at 15 players, so if there aren't enough spaces open then we're sorry
